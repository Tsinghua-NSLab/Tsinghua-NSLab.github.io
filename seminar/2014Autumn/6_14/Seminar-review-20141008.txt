Presentation[edit]Motivation
A novel framework to achieve both accountability and privacy
Previous efforts offer stronger guarantees for one of the twoAccountable and Private Internet Protocol (APIP)
The key point: the source address
The five distinct roles of source address are:
return address
sender identity
error reporting
flow ID
accountabilityThe APIP framework
splits the source address into two separate fields:
accountability address
return address
introduces delegated accountability
specialized companies as delegates
source domains as delegate
employs hidden return address
end-to-end encryption
address translationDelegated accountability
Sender -> Delegate: brief(P)
Verifier -> Delegate: verify(P)
Receiver -> Delegate: shutoff(P)Discussion[edit]Prof. Xue: 有没有实际的部署？
